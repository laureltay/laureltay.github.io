# Images Directory

Place your blog images here.

## Hero Image

Add a file named `hero.jpg` (or `hero.png`) to display as the main header image on your blog's home page.

- Recommended size: 1200x400 pixels
- Supported formats: `.jpg`, `.png`, `.webp`

## Blog Post Images

Add any images you want to use in your blog posts here. Reference them in markdown like:

```markdown
![Description](/images/your-image.jpg)
```

## Examples

```
public/images/
  ├── hero.jpg              # Main blog header image
  ├── my-photo.jpg          # Personal photo
  ├── screenshot.png        # Project screenshot
  └── project-demo.gif      # Animated demo
```
